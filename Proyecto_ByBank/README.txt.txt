PROYECTO BY_BANK
___ES
En este proyecto realizamos un inicio de plataforma bancaria. 
Utilizamos este proyecto para realizar un primer programa que demuestre la ejercitación

___EN
In this project, we are carrying out a banking platform startup. 
We use this project to realize a first program that demonstrates the training in JAVA JRD and JDK. 
We also work with objects, in the chapter "Understanding Object Orientation".
